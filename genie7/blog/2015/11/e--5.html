e커머스 데이터 파헤치기-5편

Posted on November 9, 2015

e커머스 매출을 올리는 산수(算數) - 매출을 올리는 방법 5

오늘도 역시 달리는 버스 안이다. 그래도 요즘은 집에서 랩탑 열고 이것저것 하는 분위기를 만들어가고 있다. 머 딱히 결과가 있진 않지만, 그래도 리모컨 들고 TV만 보는 것보다는 낫지 않을까 싶다. 그래도 여전히 글쓰기는 분위기는 안 만들어지고, 버스 안에서 쓰는 글이 왜 이리 편한 걸까 ㅋㅋㅋ
지난 글에서도 언급했듯이 이번 글은 "고객"을 메인 주제로 한 매출 올리는 방법이다.

이번부터 사용되는 매출액 계산 공식은 지난 2편부터 4편까지 쓰였던 공식과 다른, 그렇지만 결과는 같을 수밖에 없는 등식이다. 다시 한번 살펴보자.

매출액2 = (고객1 X 구매금액1) + (고객2 X 구매금액2) + (고객3 X 구매금액3) + ... + (고객m X 구매금액m)

지난번에 상품1 ... 상품n이 있던 자리에 고객1 ... 고객n이 자리하고 있다.
매출을 올리기 위한 방법 4로 고객1의 값을 높이는 방법이 필요하나 각 고객은 한 명이 될 수밖에 없기 때문에 방법 4는 가능한 방법이 아니라고 규정하고 넘어가도록 한다. 물론 고객이 법적 구매가 불가능한 자녀가 있어서 유아용품, 아동용품, 학습지 등을 구매하는 입장이 있어 1명에 대한 쇼핑이 아니라 2명 3명분의 쇼핑을 할 수 있으나, 이건 다른 방법으로 접근해 보기로 하자.

그럼 방법 5는 무엇이었는지 다시 기억해 보자.

방법 5 : 구매금액1, 구매금액2, 구매금액3, ..., 구매금액m의 각각의 구매금액을 높인다.

방정식에서 각 고객이 구매하는 구매 금액을 높이는 방법으로 매출액을 늘리는 공식이다. 그럼 새로운 사례를 통해 방법 5가 정말 매출액이 증가하는지 확인해보자.
[사례 2]
Mr. Hyun은인터넷 쇼핑몰 "도서닷컴"을 운영하고 있다. 이 쇼핑몰은 국내에서 취급하는 모든 서적을 판매하고 있고, 매월 100명의 고객이 도서를 주문하고 있다. 하루에는 많게는 10명에서 2명 정도가 구매를 하고 있다. 계산을 단순하게 하기 위해 2015년 10월13일과 2015년 10월 14일 이틀의 매출액2를 기준으로 계산해보자. 2015년 10월 13일의 판매 현황은 다음과 같다.

고객과 주문 현황
- 김서방 : 50,000원
- 나대로 : 30,000원
- 다나까 : 40,000원
-라구요 : 100,000원
- 마동탁 : 20,000원

매출액을 계산해보면 다음과 같다.
매출액2 = (고객1 X 구매금액1) + (고객2 X 구매금액2) + (고객3 X 구매금액3) + ... + (고객m X 구매금액m)
= (김서방 X 50,000원) + (나대로 X 30,000원) + (다나까 X 40,000원) + (라구요 X 100,000원) + (마동탁 X 20,000원)
= (50,000원) + (30,000원) + (40,000원) + (100,000원) + (20,000원) [각 고객은 모두 1의 값을 가지므로]
= 240,000원

"도서닷컴"의 2015년 10월 13일의 매출액은 240,000원이다.
그럼 이제 방법 5를 적용해 보도록 하자.
방법 5는 각 고객이 주문금액을 높이도록 하는 방법이다. 각 고객은 2015년 10월 14일에 10%만큼 더 주문했다고 가정하자. 그렇다면 2015년 10월 14일의 주문 현황은 다음과 같다.

고객과 주문 현황
- 김서방 : 55,000원
- 나대로 : 33,000원
- 다나까 : 44,000원
- 라구요 : 110,000원
- 마동탁 : 22,000원

이제 2015년 10월 14일의 매출액을 계산해보자.
매출액2 =(김서방 X 55,000원) + (나대로 X 33,000원) + (다나까 X 44,000원) + (라구요 X 110,000원) + (마동탁 X 22,000원)
= (55,000원) + (33,000원) + (44,000원) + (110,000원) + (22,000원) [각 고객은 모두 1의 값을 가지므로]
= 264,000원

10% 증가된 2015년 10월 14일의 매출액은 264,000원이다.
방법 5를 적용하여 고객의 주문하는 주문금액을 높이면 매출액이 증가한다는 것을 확인했다. 여기서도 앞의 글과 마찬가지로 숫자가 높아지는 것은 확인했으나 고객의 주문금액을 높이는 방법에 대해서는 언급되지 않았다. 왜냐하면, 그건 산수는 아니기 때문이다. ㅋㅋㅋ. 그러다 걱정 말자. 방법 6까지 정리가 되고 나면, 각 방법에 대해 하나씩 찾아보려고 한다.
사실 이 글을 시작한 목적 중 하나가 온라인 쇼핑에서(온라인 쇼핑만 해당되는 것은 아니지만) 매출을 높이기 위한 많은 활동들을 하고 있지만, 실제로 그 활동들이 이 매출액을 높이는 공식 어느 값에 영향을 주는지 이해 못하는 사람들도 적지 않다고 본다. 이 공식과 영향을 주는 값을 이해해야 각종 매출 증가를 위한 활동이 적절한 기대 효과를 가지는 지 알 수 있고, 흔한 말로 삽질을 하지 않게 되는 것이다. 이미 2차방정식, 3차 방정식, 미분, 적분을 이해하고 있는 사람에게 사칙연산을 복습하는 느낌일 수도 있겠으나, 가볍게 한번 터치하고 지나가는 것도 나쁘지 않다고 본다.
이제 다음 글에서 방법 6에 대해 알아보고 나면 아주 조금 더 깊은 이야기로 갈 수 있도록 하자.
