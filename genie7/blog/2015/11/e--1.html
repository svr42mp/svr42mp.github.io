e커머스 데이터 파헤치기-1편

Posted on November 8, 2015

e커머스 매출 올리는 산수(算數) - 매출액 계산하기

기업은 목적은 이윤 추구에 있다고 한다.뭐, 개인적으로 이 목적에 동의하지는 않지만 월급 받고 살다 보니 그냥 그렇다고 해 둬야지 뭐. 나중에 내가 회사를 차리게 되면, 난 나와 직원들이받아야 하는 높은급여를정해 놓고 나머지는 다양한 방식으로 사회에 환원할 거다. 그럴 기회나 생기면 다행이겠지만...

본론으로 돌아가서e커머스 비즈니스는상품을 최대한 많이 팔아서 매출액을 올리는 것이 e커머스 비즈니스의 가장 기본 목표이다. 주간, 월간, 연간 매출이 있을 수 있으나 편의상 하루에 대한 부분만 고려해 보자.매출액을 계산하는 공식은 다음과 같다.

매출액1 =(상품1 X 판매수량1) + (상품2 X 판매수량2) + (상품3 X 판매수량3) + ... + (상품n X 판매수량n)

여러 종류의 상품들이 다양하게 팔리기 때문에 상품의 종류는 n가지, 그리고 각 상품이 팔리는 수량도 n가지가 존재하겠네요. 그런데 이 공식을 다른 방식으로도 계산할 수 있습니다.

매출액2= (고객1 X 구매금액1) + (고객2 X 구매금액2) + (고객3 X 구매금액3) + ... + (고객m X 구매금액m)

각각의 고객이 구매한 금액을 모두 더하는 방식인 거죠. 사실 여기서 고객1, 고객2, 고객3, ..., 고객m은 모두 1이 됩니다. 왜냐하면 각각의 고객이라 함은 1명의 고객을 뜻하니까요.
결과적으로 매출액1과 매출액2는 동일한 금액이 됩니다. 그냥 편하게 매출액(=매출액1=매출액2)이라고 부르기로 하죠.

매출액1을 높이기 위해서는 어떤 방법이 있을까요? 실제로 가능한지 아닌지는 따지지 않고, 산수(算數)로만 생각해보도록 하지요.

- 방법 1 :상품1, 상품2, 상품3, ..., 상품n의 각각의 상품의 가격을 높인다.
- 방법 2 : 판매수량1, 판매수량2, 판매수량3, ..., 판매수량n의 각각의 판매수량을 증가시킨다.
- 방법 3 : n은 더 큰 수가 될 수 있도록 판매되는 상품을 늘린다.

또 있을 수도 있겠지만, 머리가 아플수 있으니 간단하게 여기까지만,
그러면, 매출액2를 높이기 위한 방법도 정리해 볼까요? 앞의 식에서 해봤으니 이건 좀 쉽겠지요?

- 방법4 : 고객1, 고객2, 고객3, ..., 고객m의 각각의 고객의 값을 높인다. 응? 아니다 이건 아니다. 왜냐하면 고객1, 고객2, 고객3, ..., 고객m은 모두 1이라고 앞에서 언급했다. 1이라고 해놓고 더 큰 값을 만들어 낼 수는 없으니까!
- 방법5 : 구매금액1, 구매금액2, 구매금액3, ..., 구매금액m의 각각의 구매금액을 높인다.
- 방법 6 : m을 더 큰 수가 될 수 있도록 더 많은 고객이 구매할 수 있도록 한다.

이것도 머리가 아플 수 있으니 여기까지만 하자.

위에서 말도 안 되는 방법 4를 제외하고, 우리는 매출을 높이기 위한 방법을 찾아 내었다.
이제 실행만 하면 된다.
